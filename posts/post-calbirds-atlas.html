Supporting the California Bird Atlas

Published: March 22, 2026

In January 2026 an exciting new 5-year project has started: the California Breeding Bird Atlas (CA BBA). This massive effort, supported by a custom version of eBird, aims to produce the first statewide Breeding Bird Atlas.
Breeding Bird Atlases provide critical information and tools for future bird conservation and monitoring. Progress and other information about the project can be found HERE.

The CA BBA divides California into 16,527 ‘atlas blocks’—roughly 3×3 mile squares each. Atlas blocks are based on 7.5′ USGS Topo Quads that are broken up into 6 smaller blocks. Each block is named with the name of the topo quad followed by the position of the block within the quad.

A common issue for contributors to the project is that it is hard to check which blocks have poor coverage while in the field. For this purpose I created a small web-based app that shows your location relative to the Atlas grid and provides basic coverage statistics. This self-funded activity is meant to support the project until a more comprehensive tool is released.

Try out the development version of this web-based app which should work in any mobile browser.
